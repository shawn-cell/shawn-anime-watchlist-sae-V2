Shawn anime-watchlist.SAE — Version 4.1 Firebase Upgrade Pack

What’s inside
- Multi-page anime dashboard with Ayanokoji-style dark UI
- Local demo mode + Firebase-ready toggle
- Auth page
- Watchlist page
- Profile page
- Community chat
- Shawn Jr anime assistant
- Live anime search via Jikan API
- GitHub Pages friendly structure

Quick start
1. Upload the whole folder to a GitHub repo.
2. Open js/firebase-config.js
3. Set useFirebase to false for demo mode OR true and paste your Firebase config.
4. Open index.html locally for testing, or deploy to GitHub Pages.

Recommended next step after upload:
- Connect Firebase Authentication + Firestore
- Add your Firebase domain to auth authorized domains
